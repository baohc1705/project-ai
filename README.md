# Ứng dụng AI nhận diện cử chỉ tay tăng giảm ánh sáng màn hình và âm thanh
--- 

## Yêu cầu:
- Python 3.11

---

## Command 

### 1. Clone
```
git clone https://github.com/baohc1705/project-ai.git

```

### 2. Tạo môi trường ảo
```
py -3.11 -m venv .venv
venv\Scripts\activate

```

### 3. Cài các gói thư viện

```
pip install opencv-python
pip install mediapipe==0.10.9
pip install screen-brightness-control
pip install pycaw
pip install comtypes
pip install numpy
pip install keyboard
```

### 4. Run

```
python main.py

```
